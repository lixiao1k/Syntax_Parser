# Syntax_Parser

### 上下文无关语法为：

```
"E->GC",
"C->AC",
"C->ε",
"A->+G",
"A->-G",
"T->FD",
"D->BD",
"D->ε",
"B->*F",
"B->/F",
"F->(E)",
"F->i"
```

### 输入：

输入语句为算术表达式，包括+-*／和（），不能包含其他字符，如 “;” 或空白符

输入语句放入input/Input.txt中

然后点击运行即可看到控制台输出解析过程。

输入实例：124+23+12*32+(23/34)

####使用了上次自己写的词法分析器，略作改动，只分析算术表达式
